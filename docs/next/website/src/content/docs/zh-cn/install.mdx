---
title: 安装 Herdr
description: 构建、安装并验证 kazuph/herdr fork。
---

kazuph/herdr fork 从源码安装。也可以明确选择 [fork releases](https://github.com/kazuph/herdr/releases) 中的二进制。不要把上游发布二进制、`herdr.dev` 安装器、Homebrew、mise 或自动 release download 用于这个 fork runtime。

## 安装

构建 fork 并安装生成的二进制:

```bash
git clone https://github.com/kazuph/herdr
cd herdr
just build
install -m 755 target/release/herdr ~/.local/bin/herdr
```

在 macOS 本地开发安装中,复制后对二进制进行签名:

```bash
codesign -s - -f ~/.local/bin/herdr
```

如果要测试 alternate binary,可以把同一个构建复制成 `herdrx` 等其他名称。该二进制会根据文件名使用独立的 config、state、socket 和 session namespace。

## 验证

启动 Herdr:

```bash
herdr
```

如果 shell 找不到 `herdr`,重启终端,或检查安装目录是否在 PATH 上。

## 更新

这个 fork 禁用 self-update。更新时从 fork checkout 重新构建,并显式替换本地二进制:

```bash
just build
install -m 755 target/release/herdr ~/.local/bin/herdr
```

正在运行的 Herdr server/client 重启是单独操作。不要因为构建了新二进制就隐式停止或替换正在运行的 runtime。

## 系统要求

需要能构建此源码树的 Rust toolchain,以及目标平台所需的系统库。
